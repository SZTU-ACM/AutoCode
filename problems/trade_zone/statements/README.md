# 繁荣的商业帝国

## 题目描述

给定一个包含 $n$ 座城市的王国。这些城市通过 $n-1$ 条双向商道相连，构成了一个连通的网络结构（即一棵树）。
树上的每座城市 $v$ 都有自己的盈亏状态。有些城市经济繁荣，能够产生收益（如果城市 $v$ 是繁荣的，则 $a_v = 1$）；有些城市经济萧条，会造成亏损（如果城市 $v$ 是萧条的，则 $a_v = 0$）。

你需要对于每座城市 $v$，解决如下问题：在所有包含城市 $v$ 的贸易区中，繁荣城市数量与萧条城市数量的最大差值是多少？
王国的贸易区必须是原网络的一个连通子图。更正式地说，如果你选择的贸易区包含 $cnt_p$ 座繁荣城市和 $cnt_l$ 座萧条城市，你需要最大化总收益（即 $cnt_p - cnt_l$）。

## 输入格式

输入的第一行包含一个整数 $n$（$2 \le n \le 2 \cdot 10^5$），表示王国中城市的数量。

第二行包含 $n$ 个整数 $a_1, a_2, \dots, a_n$（$0 \le a_i \le 1$），其中 $a_i$ 表示第 $i$ 座城市的盈亏状态（1 表示繁荣，0 表示萧条）。

接下来的 $n-1$ 行，每行描述一条商道。第 $i$ 条商道由两个整数 $u_i$ 和 $v_i$ 表示，连接城市 $u_i$ 和 $v_i$（$1 \le u_i, v_i \le n, u_i \ne v_i$）。

保证给定的商道构成一棵树。

## 输出格式

输出 $n$ 个整数 $res_1, res_2, \dots, res_n$，其中 $res_i$ 表示在所有包含城市 $i$ 的连通贸易区中，能够获得的最大总收益。

## 输入输出样例 #1

### 输入 #1

```
9
0 1 1 1 0 0 0 0 1
1 2
1 3
3 4
3 5
2 6
4 7
6 8
5 9
```

### 输出 #1

```
2 2 2 2 2 1 1 0 2
```

## 输入输出样例 #2

### 输入 #2

```
4
0 0 1 0
1 2
1 3
1 4
```

### 输出 #2

```
0 -1 1 -1
```

## 说明/提示

在第二个样例中，对于城市 $2, 3, 4$，它们对应的最优贸易区分别只包含它们自己本身。而对于城市 $1$，其最优贸易区则是同时包含城市 $1$ 和 $3$ 的连通子图。
