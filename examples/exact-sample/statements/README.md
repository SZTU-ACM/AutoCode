# Exact Sample

## 时间限制与空间限制

- 时间限制：2 秒
- 空间限制：256 MB

## 题目背景（可选）

本题为标准精确输出题示例，用于演示题面结构。

## 题目描述

给定一个长度为 `n` 的整数数组 `a`，请输出数组中所有元素之和。

## 输入格式

第一行输入一个整数 `n`，表示数组长度。  
第二行输入 `n` 个整数 `a_1, a_2, ..., a_n`。

数据范围：

- `1 <= n <= 2 * 10^5`
- `-10^9 <= a_i <= 10^9`

## 输出格式

输出一个整数，表示数组元素之和。

## 样例

### 样例输入 #1

```text
5
1 2 3 4 5
```

### 样例输出 #1

```text
15
```

### 样例输入 #2

```text
4
-3 1 2 -5
```

### 样例输出 #2

```text
-5
```

## 说明

样例 #2 展示了数组中包含负数时，结果依然是按题意直接求和。

在构建本题的 brute 时，应直接模拟“对所有元素求和”的原始约束，不要引入额外筛选或子集选择假设。
