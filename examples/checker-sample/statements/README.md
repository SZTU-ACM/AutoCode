# Checker Sample

## 时间限制与空间限制

- 时间限制：2 秒
- 空间限制：256 MB

## 题目背景（可选）

本题为特殊判题示例，答案不唯一。

## 题目描述

给定一个整数 `n`，请输出任意一个满足 `x + y = n` 的整数对 `(x, y)`。

## 输入格式

输入一行，包含一个整数 `n`。

数据范围：

- `1 <= n <= 10^9`

## 输出格式

输出一行两个整数 `x y`。  
只要满足 `x + y = n`，即判为正确。

## 样例

### 样例输入 #1

```text
10
```

### 样例输出 #1

```text
3 7
```

### 样例输入 #2

```text
10
```

### 样例输出 #2

```text
0 10
```

## 说明

本题答案不唯一，样例 #1 与样例 #2 都是合法输出。实际评测需使用 checker 判断。
